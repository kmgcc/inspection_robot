# 3.1 Plan Implementation Plan

**前置状态：** 0.1 共享契约已完成，后续开发以 docs/api_contract.md 为准。

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the dashboard, demo controls, deployment/run documentation, and presentation evidence flow so teammates can operate and demonstrate the project reliably.

**Architecture:** This plan owns the user-facing web surface and project operation docs. It consumes the shared status/event contract from `docs/api_contract.md`, keeps simulation controls as a fallback, and prepares the final defense/demo workflow. It does not implement robot drivers or business rules.

**Tech Stack:** Flask templates, plain HTML/CSS/JavaScript, shell scripts, Markdown documentation.

## 串行关系判断

页面骨架、README、演示文档可以独立开始；最终 API 字段接线使用 `docs/api_contract.md`。
