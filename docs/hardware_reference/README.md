# Hardware Reference

This folder keeps local hardware reference material for the Raspberry Pi 5 car project.

## Folders

- `raspberry_pi_5/`: Raspberry Pi 5 hardware, GPIO, I2C, and Python GPIO stack references.
- `mpu6050_module/`: Local MPU6050 module documents copied from the downloaded vendor package.

## Current Priority

Use these files first when wiring and validating the MPU6050:

1. `raspberry_pi_5/raspberry-pi-5-gpio-i2c-quick-reference.md`
2. `mpu6050_module/README.md`
3. `raspberry_pi_5/pinout-xyz-i2c.html`
4. `raspberry_pi_5/raspberry-pi-5-product-brief.pdf`
5. `raspberry_pi_5/rp1-peripherals.pdf`

## Notes

- The Raspberry Pi official HTML pages blocked direct command-line download with HTTP 403 during collection, so the key project-specific facts from those pages are preserved in Markdown notes in this folder.
- The raw source PDFs and locally supplied MPU6050 module files are kept unchanged.
