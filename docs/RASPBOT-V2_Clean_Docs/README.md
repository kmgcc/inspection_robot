# RASPBOT-V2 AI视觉小车 - 项目关联资料索引

本目录是为“固定仓库场景下的麦克纳姆轮巡逻与货架巡检系统”保留的官方文档与源码精简集。它不包含全部原厂课程，只保留本项目可能用到的硬件、运动、视觉、告警、远程部署和 ROS2 资料。

## 目录结构说明

- **03.小车基础教程/**
  - `00.开发前的准备/开发前的准备.pdf`：介绍如何关闭小车开机自启动大程序，避免硬件资源冲突。
  - `01.蜂鸣器控制/`：蜂鸣器驱动说明，用于异常和障碍提示。
  - `03.RGB控制/`：RGB 灯条控制说明，用于正常、障碍、异常状态提示。
  - `04.舵机控制（云台控制）/`：二维舵机云台控制说明，可作为摄像头姿态调整参考。
  - `06.超声波测距打印/`：超声波测距模块使用说明，用于前方障碍检测。
  - `07.四路巡线模块状态打印/`：红外巡线传感器使用说明，在新版方案中主要作为黑胶带禁区/边界兜底参考。四路状态要尽量保留方向信息，用于判断哪一侧先压到禁区边界。
- **05.小车综合玩法/**
  - `1.小车红外巡线/`：介绍基础黑线检测和低速运动逻辑；新版不把它作为主巡逻路线，只作传感器参考。
  - `4.红外巡线+超声波避障/`：介绍检测到障碍后停车和恢复的逻辑，可作超声波障碍等待参考。
- **13.ROS2-机器人底盘与控制/**
  - `1.机器人信息发布/`：介绍底盘键盘控制和底层 ROS2 交互命令。
- **14.ROS2-opencv系列课程/**
  - `9.AprilTag标签码识别/`：介绍 TAG36H11 的识别方法及 API，是货架和物品标签识别的主参考。本项目已经确认现有方块上的码属于 AprilTag TAG36H11。
  - `10.AprilTag标签码追踪/`：介绍标签识别后的摄像头云台追踪逻辑，可作为侧向扫描稳定性的参考。
  - `11.AprilTag标签码跟随/`：介绍标签识别后的底盘位置跟随控制，可作为对准货架时的参考。
- **15.开发环境搭建/**
  - `4.远程访问/`：SSH/VNC 远程连接树莓派的方法。
- **17.Docker/**
  - `5、进入小车的docker容器/`：介绍如何进入机器人容器运行 ROS2 Humble 节点。
- **19.程序源码汇总/**
  - `03.Basic_car_course/`：基础课程 Jupyter 笔记本源码，包括蜂鸣器、RGB、电机、超声波、巡线传感器等。
  - `04.Car_motion_control/`：麦克纳姆轮运动控制源码，包含前后、斜向、横移、旋转，是新版运动控制主依据。
  - `05.Comprehensive_gameplay/`：综合玩法源码，其中超声波避障和画地为牢可作为障碍等待、禁区兜底参考。
- **20.硬件资料/**
  - `树莓派与扩展板i2c通信协议/树莓派5麦轮小车协议.xlsx`：扩展板 I2C 寄存器通信协议，包含蜂鸣器、灯光、舵机和底盘电机寄存器地址，供无 ROS 的单进程 Python 控制开发使用。

## 新版项目中的资料用法

本项目的主链路是固定地图路径规划、麦轮执行、超声波避障、侧向视觉货架扫描和异常上报。对应资料关系如下：

| 功能 | 首选资料 |
|---|---|
| 前进/后退/横移/旋转 | `19.程序源码汇总/04.Car_motion_control/` |
| 前方障碍检测 | `03.Basic_car_course/6.Ultrasonic distance measurement.ipynb` |
| 障碍等待与保守绕行 | `05.Comprehensive_gameplay/2.ultrasonic_obstacle_avoidance.ipynb` |
| 黑胶带禁区兜底 | `05.Comprehensive_gameplay/5.car_geofencing.ipynb` |
| 货架/物品标签识别 | `14.ROS2-opencv系列课程/9.AprilTag标签码识别/` |
| 货架 OCR 复核 | 可基于 OpenCV 取图后接 OCR 库；只作补充，不替代 AprilTag |
| 物品颜色块检测 | OpenCV HSV 阈值；用于复核和演示 |
| 现场声光告警 | `03.Basic_car_course/1.Buzzer driver.ipynb`、`3.RGB Light bar test.ipynb` |
| 远程部署和调试 | `15.开发环境搭建/4.远程访问/`、`17.Docker/5、进入小车的docker容器/` |

二维码识别、YOLO 或其他 AI 视觉课程可作为备选路线或加分项；主链路优先使用 AprilTag TAG36H11 提供稳定身份。货架标签采用“上方大号货架号、下方 AprilTag、底部数字脚注”，OCR 货架号只作复核。物品标签采用“颜色块、简化图像、文字、AprilTag、底部数字脚注”，颜色、文字和图像都纳入检测证据，但不单独决定物品身份。

## 移出内容说明

为了使 Git 仓库轻量化，以下非必要文件已移出，应在本地或网盘保留：

1. 视频文件：所有 `.mp4` 教学视频。
2. 虚拟机及镜像：大体积 ROS2 虚拟机与出厂镜像。
3. 大模型及 AI 相关完整课程：本项目只把 LLM 作为告警后处理加分项，不依赖完整课程资料。
4. OpenCV 与机器学习基础大目录：当前只保留项目直接需要的视觉标签识别资料。
5. 硬件 3D 模型：项目不需要加工或 3D 打印修改车身。
