# RASPBOT-V2 AI视觉小车 - 极简项目关联文档

本项目是针对“基于 AprilTag 的巡线式物品巡检、异常上报与人工回收确认系统”项目特化精简后的官方文档与源码库，**总体积仅约 11.5 MB**，非常适合直接上传到 Git 仓库，避免大文件引起的历史体积膨胀。

## 目录结构说明

- **03.小车基础教程/**
  - `00.开发前的准备/开发前的准备.pdf` - 介绍如何关闭小车开机自启动大程序（学前必看，避免硬件资源冲突）。
  - `01.蜂鸣器控制/` - 蜂鸣器驱动说明。
  - `03.RGB控制/` - RGB灯条控制说明。
  - `04.舵机控制（云台控制）/` - 二维舵机云台控制说明。
  - `06.超声波测距打印/` - 超声波测距模块使用说明。
  - `07.四路巡线模块状态打印/` - 红外巡线传感器使用说明。
- **05.小车综合玩法/**
  - `1.小车红外巡线/` - 介绍基础巡线控制算法与代码逻辑。
  - `4.红外巡线+超声波避障/` - 介绍巡线过程中如何通过超声波停车及恢复。
- **13.ROS2-机器人底盘与控制/**
  - `1.机器人信息发布/` - 介绍底盘键盘控制和底层ROS2交互命令。
- **14.ROS2-opencv系列课程/**
  - `9.AprilTag标签码识别/` - 介绍 TAG36H11 的识别方法及 API。
  - `10.AprilTag标签码追踪/` - 介绍标签识别后的摄像头云台追踪逻辑。
  - `11.AprilTag标签码跟随/` - 介绍标签识别后的底盘位置跟随控制。
- **15.开发环境搭建/**
  - `4.远程访问/` - SSH/VNC 远程连接树莓派的方法。
- **17.Docker/**
  - `5、进入小车的docker容器/` - 介绍如何进入机器人容器运行 ROS2 Humble 节点。
- **19.程序源码汇总/** (精简解压版源码)
  - `03.Basic_car_course/` - 基础课程 Jupyter 笔记本源码（蜂鸣器、RGB、电机、超声波等）。
  - `04.Car_motion_control/` - 底盘麦克纳姆轮运动控制 Jupyter 笔记本源码。
  - `05.Comprehensive_gameplay/` - 巡线和避障核心 Jupyter 笔记本源码。
- **20.硬件资料/**
  - `树莓派与扩展板i2c通信协议/树莓派5麦轮小车协议.xlsx` - 扩展板的 I2C 寄存器通信协议（包含直接控制蜂鸣器、灯光、舵机和底盘电机的寄存器地址，供无ROS的单进程Python控制开发使用）。

## 移出内容说明（需在本地或网盘保留）

为了使 Git 仓库轻量化，以下非必要文件已全部移出：
1. **视频文件**：所有 `.mp4` 教学视频 (约 5.6 GB)。
2. **虚拟机及镜像**：18 GB 虚拟机 (`ros2_VM.rar`) 与 15 GB 镜像文件 (`Raspbotv2AI-20250820.zip`)。
3. **大模型及AI相关**：大模型开发与案例目录、AI视觉识别（YOLOv5、人脸、手势等，约 60 MB PDF 文件）。
4. **OpenCV与机器学习基础**：OpenCV 几何变换、图像美化和手写数字识别（约 55 MB，且包含大量图像和 SVM 样本库）。
5. **硬件3D模型**：65.94 MB 的 STEP 格式 3D CAD 模型（由于项目不需要加工或3D打印修改车身，故移出）。
